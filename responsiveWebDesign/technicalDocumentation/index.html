Introduction

This tutorial is meant to introduce the main elements of the Python programming language. Python is an extremely easy language to pick up and its syntax is straightforward and readable. To follow this tutorial you should be familiar with the basic concepts of programming. Through the following tutorial, pieces of code with three less-than signs >>> refer to input to the Python interpreter. Pieces of code without >>> refer to output from the Python interpreter.

Hello, World!

Python probably has a simpler Hello World program than any other programming language! 

          >>> print("Hello, World!")
          Hello, World!
Variables

Variables are very simple to declare in Python. Unlike many other programming languages, the data type of a variable does not need to be specified when declaring variables. Here is an example of how to declare a variable and then print it to the console: 

          >>> x = 7
          >>> print(x)
          7
Data Types

In Python there are 5 main data types. These are:

  • Numbers
  • Strings
  • Lists
  • Tuples
  • Dictionaries

To check the type of a variable in Python, use the type() function. See special reference types for other reference types.

Numbers

There are four types of numbers in Python, these are:

  • int - for integers
  • long - for long integers
  • float - for floating point numbers
  • complex - for complex numbers

Operations that can be used on numbers include addition +, subtraction -, multiplication *, division / and modulo %. Note that division will always return a float.

Strings

Strings are created in Python by enclosing text in either single quotes '' or double "" quotes. Strings can be concatenated using the + symbol.

Lists

Lists in Python are similar to arrays in other languages. They are used to store multiple pieces of data. Lists are declared using square brackets [], for example: 

          >>> aList = [1,2,3,4,5]

Lists are ordered which means an individual element can be accessed based on its position in the list. For example, to access the 2nd element in the list above (remember that the element count starts at 0): 

          >>> print(aList[1])
          2

You can also update list elements in a similar way: 

          >>> aList[3] = 10
          >>> print(aList[3])
          10

The size of a list will change automatically as new elements are added/removed. See list methods for methods that can be used on lists.

Tuples

Tuples are similar to lists but they are immutable and unchangeable. This means that elements cannot be updated or deleted. It also means that the size of a tuple is fixed. Tuples are declared using round brackets ().

Dictionaries

Dictionaries are unordered collections which store key : value pairs. They are declared using curly brackets {}. Values are accessed by referencing the key. This is best illustrated through an example: 

          >>> myDict = {'Age' : 21, 'Job' : 'Student'}
          >>> print(myDict['Age'])
          21

The entries to the left of the colon are the keys, and the entries to the right are the values.

If statements

if statements are pieces of code that execute only if a condition is true. They can optionally be used in conjuction with else if (if a different condition is true) or else (if the condition is not true) statements. An example of an if statement in Python is: 

          if (x < 10):
            print('x is less than 10')
          else if (x == 10):
            print('x equals 10')
          else:
            print('x is greater than 10')
if statements are commonly used with the logical operators and and or (sometimes denoted as && and || in other languages).
